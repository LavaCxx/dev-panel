## [0.1.0] - 2026-01-25

### Added

- 多项目管理：在单窗口内管理多个前端项目
- 分屏视图：Dev Server 日志 + 交互式 Shell
- 命令面板：快速执行 npm scripts 和自定义命令
- 自动检测 `package.json` 脚本
- 包管理器智能检测（npm/yarn/pnpm/bun）
- 进程冻结/恢复功能（macOS/Linux）
- 项目别名支持
- 配置持久化到 `devpanel.json`
- Catppuccin Mocha 主题
- 完整终端支持（ANSI 颜色、Starship 等）
- 鼠标支持（点击、滚轮）
- 中英文界面切换
- 数字键 1-9 快速切换项目
- Dev Terminal 滚动查看历史 log

### Platforms

- macOS (Intel & Apple Silicon)
- Linux (x86_64 & ARM64)
- Windows (x86_64)
